Syarat & Ketentuan

1. Ketentuan Umum

Dengan accessing and using EliteZone services, you agree to be bound by these Terms and Conditions. If you do not agree, please do not use our services.

2. Transaksi dan Pembayaran

Pastikan User ID dan Server ID yang Anda masukkan sudah benar. Kesalahan input sepenuhnya menjadi tanggung jawab pembeli. EliteZone tidak akan memberikan pengembalian dana (refund) atas kelalaian pembeli. Pembayaran harus dilakukan sesuai dengan nominal dan metode yang telah dipilih sebelum batas waktu habis.

3. Waktu Proses

Proses top-up umumnya membutuhkan waktu 1-5 menit. Pada kondisi tertentu (seperti gangguan dari server game pusat), proses dapat memakan waktu hingga 24 jam. Kami berkomitmen untuk menyelesaikan pesanan Anda secepat mungkin.

4. Kebijakan Pengembalian Dana (Refund)

Refund hanya akan dilakukan jika pesanan gagal diproses oleh sistem karena stok kosong atau gangguan server yang berkepanjangan, dan dana sudah masuk ke rekening kami.

Kebijakan Privasi

1. Pengumpulan Informasi

Kami mengumpulkan informasi yang Anda berikan secara langsung kepada kami, seperti Nomor WhatsApp, User ID Game, dan Nama Lengkap (jika mendaftar akun). Informasi ini digunakan semata-mata untuk memproses transaksi dan mengirimkan bukti pembelian (invoice).

2. Penggunaan Informasi

Data yang kami kumpulkan akan digunakan untuk memverifikasi dan mengeksekusi pesanan top-up Anda, contacting you if terdapat kendala pesanan, serta untuk mencegah penipuan/aktivitas mencurigakan.

3. Perlindungan Data

EliteZone berkomitmen melindungi data pribadi Anda. Kami tidak akan pernah menjual, menyewakan, atau membagikan nomor WhatsApp Anda kepada pihak ketiga untuk tujuan pemasaran tanpa izin Anda.
